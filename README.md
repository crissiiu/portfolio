# Portfolio Website

This is my personal portfolio website built with **HTML, CSS, and Vanilla JavaScript**.

The website supports **multiple languages** with:
- Vietnamese as the default language
- English version under `/en`

The project focuses on **SEO, performance, and clean architecture**, following real-world development practices without heavy frameworks.

## 🌍 Languages
- Vietnamese (default): `/`
- English: `/en`

## ✨ Features
- Semantic HTML5
- Multi-language SEO-friendly structure
- Clean and modular CSS architecture
- Lightweight JavaScript
- Optimized for Core Web Vitals
- Fully responsive design
- Free deployment ready

## 📁 Project Structure
- `index.html`: Vietnamese homepage
- `en/index.html`: English homepage
- `assets/`: Static assets
- `docs/`: Project documentation

## 🚀 Deployment
- GitHub Pages
- Cloudflare Pages
