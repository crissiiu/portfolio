# CSS Architecture

CSS files are separated by responsibility to improve readability and maintainability.

## Files
- `base.css`: Global styles, CSS variables, reset, typography
- `layout.css`: Page layout (header, footer, grid, sections)
- `components.css`: Reusable UI components (buttons, cards, badges)
- `responsive.css`: Media queries for responsive design

This structure follows real-world CSS architecture practices.
